## 🌟 نمای کلی

تولیدکننده پروفایل آیفون TECSO یک مجموعه جامع از ابزارهای حرفه‌ای است که برای ایجاد پروفایل‌های منحصر به فرد پیکربندی موبایل برای کاربران آیفون در ایران طراحی شده است. راه‌حل ما به حفظ اتصال قابل اعتماد سلولار از طریق تولید شناسه‌های منحصر به فرد و فایل‌های کامل پیکربندی موبایل کمک می‌کند.

### ✨ ویژگی‌های کلیدی

- 🆔 **تولیدکننده UUID** - ایجاد شناسه‌های منحصر به فرد برای جلوگیری از مسدود شدن پروفایل
- 📱 **تولیدکننده قالب** - ساخت فایل‌های کامل پیکربندی موبایل
- 🎨 **رابط کاربری مدرن** - تم تیره حرفه‌ای با لهجه‌های نئونی
- 🔧 **پشتیبانی چند اپراتور** - همراه اول، ایرانسل، رایتل، تمام شبکه‌ها
- 💾 **صادرات آسان** - ذخیره به عنوان فایل‌های `.mobileconfig` برای نصب روی آیفون
- 🚀 **اپلیکیشن‌های مستقل** - نیازی به نصب نیست، فقط اجرای فایل‌های اجرایی

## 🛠️ ابزارهای شامل

### 1. 🆔 تولیدکننده UUID
تولید شناسه‌های منحصر به فرد برای پروفایل‌های موبایل برای جلوگیری از مسدود شدن زمانی که چندین دستگاه از همان پیکربندی استفاده می‌کنند.

**ویژگی‌ها:**
- ⚡ تولید فوری UUID
- 📋 کپی با یک کلیک به کلیپ‌بورد
- 🔑 فرمت‌های استاندارد و HEX
- 🎯 رابط حرفه‌ای

### 2. 📱 تولیدکننده قالب
ایجاد پروفایل‌های کامل پیکربندی موبایل با تنظیمات قابل تنظیم.

**ویژگی‌ها:**
- 📡 پشتیبانی چند اپراتور (همراه اول، ایرانسل، رایتل)
- ⚙️ تنظیمات قابل پیکربندی APN
- 🔧 گزینه‌های تایید هویت پیشرفته
- 💾 عملکرد ذخیره/بارگذاری
- 🎨 پیش‌نمایش زنده XML تولید شده

### 3. 🚀 لانچر حرفه‌ای
مرکز مرکزی برای دسترسی به تمام ابزارهای TECSO با اطلاعات تماس یکپارچه.

**ویژگی‌ها:**
- 🎯 دسترسی سریع به ابزارها
- 📞 دکمه‌های تماس یکپارچه
- 🌐 لینک‌های مستقیم به منابع TECSO
- ✨ طراحی رابط مدرن

## 🎯 شروع سریع

### روش 1: استفاده از فایل‌های اجرایی از پیش ساخته شده
1. **دانلود** آخرین نسخه از مخزن ما
2. **اجرا** `TECSO-Launcher-Updated.exe` برای رابط اصلی
3. **کلیک** روی ابزاری که نیاز دارید (تولیدکننده UUID یا تولیدکننده قالب)
4. **تولید** پروفایل منحصر به فرد خود و ذخیره به عنوان `.mobileconfig`
5. **انتقال** به آیفون و نصب از طریق تنظیمات

### روش 2: اجرا از کد منبع
```bash
# کلون کردن مخزن
git clone https://github.com/Tecso-Dev/Iphone-14-15-IRAN-Anten.git

# رفتن به دایرکتوری
cd Iphone-14-15-IRAN-Anten

# تنظیم محیط پایتون
python -m venv .venv
.venv\Scripts\activate

# نصب وابستگی‌ها
pip install pyperclip

# اجرای ابزارها
python uuid-Generator.py           # تولیدکننده UUID
python template-generator.py       # تولیدکننده قالب
python launcher.py                 # لانچر اصلی
```

## 📖 راهنمای استفاده تفصیلی

### 🔧 ایجاد پروفایل موبایل

1. **شروع تولیدکننده قالب**
   ```
   اجرا: TECSO-Template-Generator-Updated.exe
   ```

2. **پیکربندی پروفایل شما**
   - اپراتور موبایل خود را انتخاب کنید
   - نام پروفایل و سازمان را وارد کنید
   - یک UUID منحصر به فرد تولید کنید یا وارد کنید
   - تنظیمات APN را پیکربندی کنید (پیش‌فرض‌ها برای اکثر موارد کار می‌کنند)

3. **تولید و ذخیره**
   - روی "⚡ تولید قالب" کلیک کنید
   - XML تولید شده را بررسی کنید
   - روی "💾 ذخیره قالب" کلیک کنید
   - به عنوان فایل `.mobileconfig` ذخیره کنید

4. **نصب روی آیفون**
   - فایل را به آیفون منتقل کنید (AirDrop، ایمیل، و غیره)
   - فایل را روی آیفون باز کنید
   - به تنظیمات → VPN و مدیریت دستگاه بروید
   - پروفایل را نصب کنید
   - داده سلولار را فعال کنید و گوشی را مجدداً راه‌اندازی کنید

### 🆔 تولید UUID‌های منحصر به فرد

```python
# هر پروفایل نیاز به یک UUID منحصر به فرد برای جلوگیری از مسدود شدن دارد
UUID: d85183f5-84b7-4895-9eb3-eb76e7cd45ab
HEX:  d85183f584b748959eb3eb76e7cd45ab
```

**چرا UUID‌های منحصر به فرد مهم هستند:**
- از تشخیص و مسدود شدن پروفایل جلوگیری می‌کند
- اجازه می‌دهد چندین دستگاه از پیکربندی‌های مختلف استفاده کنند
- اتصال قابل اعتماد طولانی مدت را تضمین می‌کند
